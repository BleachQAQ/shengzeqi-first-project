# shengzeqi-first-project

## 项目介绍
这是盛泽旗的第一个GitHub测试项目，用于学习Git版本控制、GitHub仓库管理，完成课程实践作业。本项目将记录MeetSpot智能会面推荐系统从本地复现到远程同步的全流程，积累开发与版本管理经验。

## 使用方法
1. 本地克隆仓库：在命令行执行`git clone https://github.com/BleachQAQ/shengzeqi-first-project.git`，将代码拉取到本地。
2. 提交与推送：完成代码修改后，执行`git add .`、`git commit -m "更新说明"`、`git push`，将变更同步到远程仓库。
